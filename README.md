# Muminvalo libroj

## Kometo en Muminvalo

Kiam Mumintrolo lernis ke kometo preterpasos, li kaj lia amiko Snif vojaĝis al la Observatorio sur la Soleca Montaro por konsulti la profesorojn. Laŭ la vojo, ili havis multajn aventurojn, sed la plej granda aventuro de ĉiuj atendis ilin kiam ili lernis ke la kometo rekte celis ilian amatan Muminvalo.
